PhyloCommons

PhyloCommons is a web application designed to simplify the sharing, annotation, querying, and reuse of phylogenetic trees.

The web application frontend to PhyloCommons is currently in development. The RDF treestore behind PhyloCommons can be accessed at https://github.com/bendmorris/rdf-treestore
